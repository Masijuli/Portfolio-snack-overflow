# Snack Overflow

## Overview

Snack Overflow is a food inventory management system designed to help users keep track of food stored at home. The application allows users to monitor purchased food items, expiration dates, grocery usage, and grocery lists through an easy-to-use web interface.

This project was created as a team database project focused on solving common household food management problems such as forgotten groceries, expired food, and inefficient shopping habits.

The system combines a PostgreSQL database with a Flask backend and a simple frontend interface to allow users to organize and manage food inventory in real time.

---

## Features

- Track food currently stored at home
- Monitor expiration dates
- Record food purchases and usage
- Automatically update inventory quantities
- Create and manage grocery lists
- Track frequently purchased items
- View recipes and ingredient usage
- Store grocery store information

## Installation Instructions

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
python -m pip install flask "psycopg[binary]"
```

3. Run the application

```bash
python app.py
```

4. Open the website in your browser

```text
http://127.0.0.1:5000
```

---

## Usage Instructions

Users can interact with the web application to:
- Add food items to inventory
- Track grocery purchases
- Update food quantities after usage
- Monitor expiration dates
- Create grocery lists
- Organize recipes and ingredients

The system updates the database dynamically as users interact with the application.
